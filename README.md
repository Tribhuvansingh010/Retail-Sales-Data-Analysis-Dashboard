# Retail-Sales-Data-Analysis-Dashboard

A full-stack data analytics project that transforms raw CSV data into meaningful business insights using an automated data pipeline and interactive visual dashboard.
This project simulates a real-world retail analytics system where users can upload datasets, which are then cleaned, processed, and visualized to support data-driven decision-making.

Project Overview 

This system processes retail datasets (Sales, Customers, Products) and converts them into structured insights.
It follows a complete data pipeline:
CSV Upload → Raw Data → Data Cleaning → JSON Conversion → Dashboard Visualization
The goal is to demonstrate end-to-end data engineering and frontend visualization skills.

 Features
- Upload CSV files (Sales, Customers, Products)
- Automated data cleaning using Python
- Data transformation pipeline (raw → cleaned → JSON)
- Interactive dashboard with charts:
  - Monthly Sales Trend
  - Top Products by Revenue
  - Customer Distribution
- Search and filter functionality
- Auto-refresh data updates
- Real-time data visualization using Chart.js
